课程介绍：

本课程大喵将手写每一行代码，手把手带着大家从环境搭建开发，逐步开发一个零基础入门级的实战项目；通过基础实战项目来快速学习掌握Electron的相关API知识点，以及掌握使用Electron构建客户端应用程序的流程，体验Electron客户端开发的魅力和强大；本课程最终，会把我们开发的实战项目代码，分别编译打包输出三个平台（windows、mac、linux）系统所支持的PC客户端应用。
